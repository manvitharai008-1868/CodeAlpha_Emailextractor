# Email Extractor

## Description

This project was developed as part of the CodeAlpha Python Programming Internship.

The Email Extractor is a Python automation script that extracts all valid email addresses from a text file and saves them into a separate output file. This project demonstrates the use of regular expressions (`re`) and file handling to automate a repetitive real-world task.

## Features

* Reads text from an input file
* Extracts all valid email addresses
* Saves extracted email addresses into a new text file
* Displays extracted email addresses in the terminal
* Beginner-friendly Python automation project

## Technologies Used

* Python
* Regular Expressions (`re`)
* File Handling

## How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run the following command:

python extractor.py
